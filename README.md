# KoPubDotum-Webfont
KoPub 돋움체의 웹폰트 입니다.
적용하실 사이트의 assets 폴더 내에 두 폴더를 넣으신 후, CSS만 불러오시면 됩니다.

## 폰트 종류
1. Light ― 300
2. Medium ― 400
3. Bold ― 600

## 적용 방법
### HEAD에 넣는 방법
<pre>&lt;link rel="stylesheet" type="text/css" href="/path/css/KoPubDotum.css"&gt;</pre>

### CSS에 넣는 방법
<pre>@import url('KoPubDotum.css')</pre>
